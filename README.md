## Getting Started

### Prerequisites
- Go (v1.25+ recommended)
- MySQL database

### Installation
1. Create environment file:
```bash
cp .env.example .env
```

2. Replace your own value in `.env`

3. Install Go dependencies:
```bash
go mod download
```

4. Run the server:
```bash
go run cmd/main.go
```
